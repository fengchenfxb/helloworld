# 天津公交投票专题页设计方案

## 设计风格：克莱因蓝高定风

### 设计理念
以"高端票务+公共交通"为基调，融合克莱因蓝的纯粹感与香槟金的精致感，营造正式、权威、现代化的投票活动氛围。

### 配色方案
| 角色 | 色值 | 用途 |
|------|------|------|
| 主色 | #002FA7 | 导航栏、活动主色调、按钮 |
| 辅色 | #C9A96E | 装饰线、票数高亮、强调色 |
| 背景 | #F8F6F2 | 米金白底色 |
| 卡片 | #FFFFFF | 作品卡片 |
| 文字主 | #1A1A2E | 正文 |
| 文字白 | #FFFFFF | 蓝色区域上的文字 |
| 文字金 | #C9A96E | 票数、强调 |

### 页面结构

#### 1. 顶部区域
- 深蓝色背景 (#002FA7) 延伸至状态栏
- 活动标题：天津公交创意投票
- 副标题：2024年度创意设计评选
- 金色装饰线分隔

#### 2. 投票状态卡片
- 显示剩余票数
- 实名认证状态
- 定位状态
- 今日任务进度（扫码乘车、分享活动）

#### 3. 类别切换 Tab
- 顶部 Tab 导航（顶部子Tab）
- 三个类别：我们的专列 | 出行新风貌 | 梦想候车亭
- 选中态：蓝色背景 + 金色下划线

#### 4. 作品列表
- 瀑布流布局（两列交错）
- 作品卡片：
  - 缩略图
  - 作品名称
  - 作者
  - 当前票数（金色数字）
  - 投票按钮（已投/可投状态）
  - 简介按钮

#### 5. 底部常驻按钮
- 生成拉票海报（固定在底部）
- 活动规则按钮

### 交互状态

#### 投票按钮状态
- 可投票：蓝色按钮「投票」
- 已投过：灰色按钮「已投票」
- 票数用完：提示用户完成任务

#### 票数提示文案
- 有票数：显示剩余票数
- 没票数但可做任务：引导完成任务
- 没票数且任务完成：明日再来

### 拉票海报
- 作品图 + 作品编号 + 二维码 + 拉票文案
- 一键生成并可分享至朋友圈
